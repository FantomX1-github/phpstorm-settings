Nette settings
==============

Based on [http://doc.nette.org/en/2.1/coding-standard](http://doc.nette.org/en/2.1/coding-standard).

Current setup
-------------

- tab (`\t`) as indentation character
- right margin set to 120
- uppercase `TRUE`, `FALSE` and `NULL`
- `*.phpt` files marked as PHP
- Latte plugin
